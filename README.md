# Pawtterns Website

Marketing website for Pawtterns iOS app — analytical pet digestion tracking.

## Overview

Single-page website built with React, TypeScript, and Tailwind CSS. Hosts the app introduction and publicly accessible privacy policy required by the Apple App Store.

## Design Language

Matches the Pawtterns iOS app design system:
- **Sage green** (#7FB069) primary color
- **Natural off-white** (#F5F5F0) background
- **Forest green** (#3D5A4C) text
- **Terracotta** (#E07A5F) accents

**Personality:** Trusted companion, not clinical tool. Clear, warm, focused.

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Vite
- GitHub Pages

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

Deployed automatically to GitHub Pages via GitHub Actions when changes are pushed to `main`.
